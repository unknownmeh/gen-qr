# 🚀 GEN-QR

GEN-QR is a fast, simple, and unlimited QR Code Generator built using HTML, CSS, and JavaScript.

## ✨ Features

- ⚡ Instant QR code generation
- 🔗 Supports text and URLs
- ♾️ Unlimited usage
- 📥 Download QR as PNG
- 📱 Fully responsive (mobile + desktop)
- 🎨 Clean and modern UI

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript

## 📸 How It Works

1. Enter your text or URL  
2. Click **Generate QR**  
3. View your QR instantly  
4. Download it with one click  

## ▶️ Run Locally

1. Download or clone the repository  
2. Open `index.html` in your browser  

No installation required ✅
